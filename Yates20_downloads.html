Yates+20b Downloads

Yates+20b catalogues for download

The catalogues below contain the complete output galaxy trees for the modified L-Galaxies 2020 model run on the first ten tree files of Millennium-I (see Yates+20b). These output files are in binary format, so require the following structure to be read in either python or IDL: The units are as follows:

  • masses - \(10^{10}M_{\odot}/h\) (multiply by \(10^{10}/h\) to get \(M_{\odot}\))
  • positions - \(\mathrm{Mpc}/h\) (multiply by \(h\) to get \(\mathrm{Mpc}\))
  • velocities - \(\mathrm{km/s}\)
  • star formation and accretion rates - \(M_{\odot}/\mathrm{yr}\)
  • spins - \(\mathrm{Mpc}/h \:\: \mathrm{km/s}\)

Further details about the catalogs can be found in the description of each table in the databse and in this document.
