Yates+20b Downloads

Yates+20b catalogues for download

The catalogues below contain outputs for a representative sample of tree files from the modified L-Galaxies 2020 model. These are based on the Millennium and Millennium-II dark matter simulations scaled to the PLANCK year1 cosmology (Volume_MR = \(480.3^3[(\rm{Mpc}/h)^3]\), Volume_MRII = \(96.1^3[(\rm{Mpc}/h)^3]\)). These output files are in binary format, so require the following structure to be read using either python or IDL:

The units are as follows:

  • masses - \(10^{10}M_{\odot}/h\) (multiply by \(10^{10}/h\) to get \(M_{\odot}\))
  • positions - \(\mathrm{Mpc}/h\) (multiply by \(h\) to get \(\mathrm{Mpc}\))
  • velocities - \(\mathrm{km/s}\)
  • star formation and accretion rates - \(M_{\odot}/\mathrm{yr}\)
  • spins - \(\mathrm{Mpc}/h \:\: \mathrm{km/s}\)
