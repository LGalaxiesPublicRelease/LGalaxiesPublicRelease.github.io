Workshop 2025

Basic information

Date: 15th to 17th October 2025
Location: Harnack Haus, Berlin, Germany


The workshop will be hybrid, with online participation available via zoom.


The workshop will include:
(a) invited and contributed talks
(b) discussions sessions
(c) a "hackathon" to get new users up-and-running with the L-Galaxies 2020 code

Code of conduct

All participants are expected to be professional and considerate towards others during discussions and events. The code of conduct of the EAS2025 can be used for reference.
