Project: BH/NS mergers

Status

Ongoing

Collaborators

Jess Hislop, Peter Thomas

Description

We use the Henriques+15 version of L-Galaxies, together with the BPASS SPS models, to predict the stellar black hole and neutron star merger rates for comparison with results from LIGO.
[Note: this project is essentially complete but in temporary stasis due to unresolved differences between different versions of the BPASS SPS model.]
