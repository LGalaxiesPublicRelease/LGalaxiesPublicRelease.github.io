Installing L-Galaxies: Windows

System requirements

Below, we outline how to download, install, and run L-Galaxies and it's associated components on a Windows system. The recommended storage requirements are:

To run on Millennium-I:
Minimum: ~2.5 GB (to run only on treefile 5)
Typical: ~9.2 GB (to run on treefiles 0-9)
Maximum: ~402.2 GB (to run on the full Millennium-I)

To run on Millennium-II:
Minimum: ~2.5 GB (to run on a single treefile - warning: not representative)
Typical: ~6.3 GB (to run on treefiles 40-79)
Maximum: ~186.7 GB (to run on the full Millennium-II)


For reference, this particular installation was done on the following system:
Machine: x64-based PC laptop
Operating system: Windows 10 Eduction (Build 19044)
Processor: Intel Core i9, 2500 MHz, 14 cores
RAM: 32 GB Available storage: 943 GB
