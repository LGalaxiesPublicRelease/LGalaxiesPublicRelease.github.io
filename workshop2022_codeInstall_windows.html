Installing L-Galaxies: Windows

System requirements

For a basic L-Galaxies 2020 installation, the code files, some dark matter merger-tree files, and some spectrophotometric tables are required. The total storage space requirements for these basic files are:

When run on Millennium-I:
Minimum: ~2.5 GB (to run only on treefile 5)
Typical: ~9.2 GB (to run on treefiles 0-9)
Maximum: ~402.2 GB (to run on the full Millennium-I)

When run on Millennium-II:
Minimum: ~2.0 GB (to run on a single treefile - warning: not representative)
Typical: ~6.3 GB (to run on treefiles 40-79)
Maximum: ~186.7 GB (to run on the full Millennium-II)


Note that the above estimates do not include the space required to store newly-generated output files. The size of these can vary greatly, depending on the number of treefiles used, whether "single snapshot" or "galtree" mode is chosen, and (in the single snapshot case) which redshift is outputted. A rough guide to the typical size of each output file for the L-Galaxies 2020 "Default Model" is provided here:

Outputs when run on Millennium-I:
Single snapshot mode: ~50 MB - 1.5 GB
Galtree mode: ~10-25 GB

Outputs when run on Millennium-II:
Single snapshot mode: ~25-350 MB (excluding treefile 0)
Galtree mode: ~10-25 GB


For reference, the example installation detailed below was done on the following system:
Machine: x64-based PC laptop
Operating system: Windows 10 Eduction (Build 19044)
Processor: Intel Core i9, 2500 MHz, 14 cores
RAM: 32 GB
Available storage: 943 GB
