Installing L-Galaxies: Windows

System requirements

For a basic L-Galaxies 2020 installation, the code files, some dark matter merger-tree files (i.e. "treefiles"), and some spectrophotometric tables are required. In most cases, only a handful of treefiles are required to return a representative sub-volume containing a realistic range of galaxies (i.e. to roughly reproduce the observed stellar mass function at z=0). The total storage space requirements for these basic files are:

When run on Millennium-I:
Minimum: ~2.5 GB (to run on treefile 5)
Typical: ~9.2 GB (to run on treefiles 0-9)
Maximum: ~402.2 GB (to run on the full Millennium-I, 512 treefiles)

When run on Millennium-II:
Minimum: ~2.0 GB (to run on a single treefile - WARNING: not a representative sub-volume)
Typical: ~6.3 GB (to run on treefiles 40-79)
Maximum: ~186.7 GB (to run on the full Millennium-II, 512 treefiles)


Note that the above estimates do not include the space required to store newly-generated output files. The sizes of these can vary greatly, depending on the number of treefiles used, whether "single snapshot" or "galtree" mode is chosen, and (in the single snapshot case) which redshift is outputted. A rough guide to the typical size of each output file for the L-Galaxies 2020 "Default Model" is provided here (for a full set out output files from the whole simulation, multiply by 512):

Outputs when run on Millennium-I:
Single snapshot mode: ~50 MB - 1.5 GB each
Galtree mode: ~10-25 GB each

Outputs when run on Millennium-II:
Single snapshot mode: ~25-350 MB each (excluding treefile 0)
Galtree mode: ~10-25 GB each


For reference, the example Windows installation below was done on the following system:
Machine: x64-based PC laptop
Operating system: Windows 10 Eduction (Build 19044)
Processor: Intel Core i9, 2500 MHz, 14 cores
RAM: 32 GB
Available storage: 943 GB

Step 1: Download L-Galaxies 2020

OPTION 1: Download as zip file:

  1. Go to https://github.com/LGalaxiesPublicRelease/LGalaxies2020_PublicRepository
  2. Click the green "Code" drop-down menu, and select "Download ZIP" [~42.2 MB]
  3. Use any un-zipping software (e.g. 7-Zip, or other), to extract the model into a folder called /LGalaxies2020_PublicRepository-master (hereafter, the "project root directory")
  4. Manually add an empty /output folder in the project root directory

Step 2: Download spectrophotometric tables and treefiles

Download as zip files:

  1. Download the standard SpecPhotTables [~1.6 GB]
  2. Un-zip them into a folder called /SpecPhotTables in the project root directory
  3. Download the treefiles. Choose from one of the following:
    • Treefile 5 from Millennium-I [~803 MB]
    • Treefiles 0-9 from Millennium-I [~7.5 GB]
    • All 512 treefiles from Millennium-I [~401 GB]
    • Treefiles 40-79 from Millennium-II [~4.6 GB]
    • Treefiles 40-79 from Millennium-II [~9 GB]
    • All 512 treefiles from Millennium-II [~185 GB]
