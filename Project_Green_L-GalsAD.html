Project: L-Galaxies AD

Status

Ongoing

Collaborators

Description

SAMs, such as L-Galaxies, have the well-known advantage that they can be run relatively quickly (compared to hydrodynamical simulations), meaning that robust parameter constraint techniques (e.g., MCMC) are tractable under certain circumstances. However, as physics models and spatial resolution become increasingly sophisticated in SAMs, their runtime inevitably increases, calling into question this unique advantage. To tackle this, we are developing a new optimised and highly parallelised C++ version of L-Galaxies (based on the Yates+24 physics), which uses automatic differentiation (AD) to calculate parameter gradients. AD enables the efficient and accurate evaluation of partial derivatives, allowing the direct application of fast gradient-based inference techniques, such as HMC, to constrain free parameters. L-Galaxies AD will be used to generate extensive galaxy catalogues efficiently, and also to trial directly constraining parameters on super-high-resolution hydrodynamical simulations using Bayesian inference.
